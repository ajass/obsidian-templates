# Obsidian Templates

Personal note-taking templates designed for memoir and life story writing, following the Zettelkasten method.

## Templates

- `templates/memory.md` - Capture childhood memories, events, and life moments with sensory and emotional context

## Structure

```
.
├── templates/          # Obsidian note templates
└── .opencode/agents/   # Sub-agents for this project
```

## Usage

Copy templates from `templates/` into your Obsidian vault's template folder.

## Agents

- `template-builder` - Creates new templates
- `zettelkasten-advisor` - Advises on structure and tagging
- `note-migrator` - Helps migrate existing notes
